About

This application was inspired by the need to improve the organization of our IEEE student Branch at the UWI Mona Campus. This application aims to improve the experience of branch members in several ways, such as communication, scheduling as well as feedback. This application will utilise this web page to issue messages to members.

Source code for the application can be viewed on our Github page.

View The Mobile App

The Admin Area

Administrators can create events and send messages to members for upcoming meetings and notifications. The administrators can also view created events and messages within the application.
