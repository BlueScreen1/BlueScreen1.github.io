About

This application was inspired by the need to improve the organization of our IEEE Student Branch at the UWI Mona Campus in Jamaica. This application aims to improve the experience of branch members in several ways.

These include:

  • Communication with message feeds
  • Scheduling of events and meetings
  • Knowledge of IEEE
  • Group involvement and participation
  • Feedback Methods

BlueScreen1.github.io is utilised for creation of events, messages and groups for use in the application. In addition to this, Parse.com is used to for database entry handling.

Source code for the application can be viewed in the repository from our GitHub page.

The Admin Area

Administrators can create events, create groups and send messages to members for upcoming meetings and notifications. The administrators can also view created events within the application in this area.
